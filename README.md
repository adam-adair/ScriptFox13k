# ScriptFox13k

A Starfox-inspired space shooter made for the 2021 [JS13k Games competition](https://js13kgames.com/).

## How To Play

- Move your ship with WASD.
- Rotate left/right with QE.
- Fire with Spacebar.
- Pause with P.

Try to get a high score!
